Quick fixes for the most common issues on Windows 10 & 11: installer blocked by SmartScreen, antivirus flags, missing codecs, export errors, and performance hiccups.

Common issues and fixes

Installer blocked by SmartScreen
  1. Ensure the file was downloaded from nchsoftware.com (see file properties → Digital Signatures).
  2. Double‑click vppsetup.exe. If SmartScreen appears, click More info → Run anyway.
  3. Follow the on‑screen wizard. If still blocked, right‑click → Properties → check Unblock → Apply → Run.
Antivirus flagged the installer
  • False positives can happen with new builds. Verify the source is nchsoftware.com.
  • Update antivirus definitions and retry the download.
  • If needed, temporarily allow the installer, then re‑enable protection after installation.
Re‑download from source
Installer won’t start or gets stuck
  • Right‑click the installer → Run as administrator.
  • Free up disk space (~200 MB+) and close other installers.
  • Restart Windows and try again from a clean boot.
  • Download a fresh copy of vppsetup.exe using the buttons on this page.
Missing codecs or video won’t import
  • Ensure Windows is up to date (Settings → Windows Update).
  • On Windows N editions, install the Media Feature Pack from Microsoft.
  • Transcode incompatible clips to H.264 MP4 (a widely supported format) and retry.
Export has no audio or fails at 0%
  • Try exporting a short 10‑second clip with default settings (MP4 H.264/AAC).
  • Check that your timeline audio isn’t muted and meters show activity.
  • On Windows N, install the Media Feature Pack to enable AAC codecs.
Crashes, freezes, or slow performance
  • Update your GPU drivers (NVIDIA/AMD/Intel).
  • Close background apps and large browser tabs while exporting.
  • Use proxy/low‑res preview if available; keep projects on fast local storage.
